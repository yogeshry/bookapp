# Intro to DB django-crud-example
A simple crud in django

Demo:

- [heroku](https://dashboard.heroku.com/apps/allbookapp/resources)

Used:

- Django 2
- Bootstrap 4

Get Started:

```
#in env

$ sudo pip install -r requirements.txt

$ ./manage.py migrate

$ ./manage.py runserver

#django admin

$ python manage.py createsuperuser

access /admin

```
